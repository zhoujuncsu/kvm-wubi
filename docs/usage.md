## 使用说明

### 运行方式

- 源码运行：执行 main_gui.py
- exe 运行：dist/WubiKVM_VoiceInput.exe（旁边放 corrections.json）

### GUI 操作

1. 打开程序后，在文本框内输入/粘贴要发送的内容
2. 将“按键延迟（毫秒）”设置为合适值（KVM 越慢，延迟越大）
3. 切换到目标窗口（KVM 控制台/远程桌面/记事本）
4. 按 F9 开始自动输入
5. 输入过程中按 Esc 中止

### 输入法配置（目标主机）

必须满足两点：

1. 使用五笔输入法（中文模式可输入五笔编码）
2. 开启“单字模式”

如果目标主机的 Shift 并不是中/英切换键，请把执行层的切换逻辑改为你实际使用的切换键（例如 Ctrl+Space、Alt+Shift 等）。

### corrections.json（外置）

放置位置：

- 源码运行：项目根目录（与 main_gui.py 同级）
- exe 运行：与 WubiKVM_VoiceInput.exe 同目录

示例：

```json
{
  "多": "qqu2",
  "半": "ufk2"
}
```

说明：

- 末尾数字表示候选序号；例如 qqu2 表示输入 qqu 后选第 2 个候选
- 末尾带数字时，程序不会再追加空格提交，避免多余空格
